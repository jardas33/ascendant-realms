# Art Review

This directory is a non-runtime review workspace for future visual candidates.

Nothing in this directory is loaded by the game. Runtime code must not import files from here, Phaser preload paths must not reference it, and files here are not production-approved by their presence alone.

## Rules

- Use this area only for review documents, metadata, templates, and explicitly approved future candidate files.
- Do not place files here as a shortcut to runtime integration.
- Do not copy files from here into `public/assets/` without a separate scoped goal, source/license proof, manifest validation, screenshot QA, and human approval.
- Raw candidate binaries in intake/status folders are ignored by default. Commit them only with explicit approval and matching metadata.
- Unknown-source files are not production-safe.
