# v0.12.6 Ready-To-Send Tester Message

Date: 2026-05-18

Copy, edit lightly, and send this to a tester.

```text
Hey! I am looking for quick feedback on Ascendant Realms, a small fantasy RTS/RPG browser prototype.

The goal is not to judge final art yet. A lot of visuals are still placeholder, so please ignore ugly prototype art, rough scale, missing animation polish, and unfinished VFX unless they stop you from understanding the game.

What I do want feedback on:
- Did you know what to do?
- Did commands feel responsive?
- Were objectives readable?
- Did you notice pressure warnings during combat?
- Did victory or defeat feel fair?
- Was it fun enough to keep playing or retry?

Time options:
- Quick run: about 30 minutes.
- Full run: about 2 hours, if you have time.

Please start with the route I assign you. If I do not assign one, use the Baseline Cautious route: read objectives, avoid heavy optimization, react to warnings, preserve units, and retreat if badly hurt.

After playing, please send back:
- route/profile
- time played
- missions played
- wins/losses/timeouts
- 1-5 ratings for clarity, fairness, commands, objectives, pressure noticeability, combat readability, result usefulness, and fun
- top confusing moments
- top satisfying moments
- whether pressure warnings were noticed
- one thing to fix first
- one thing not to change
- screenshots/video links if useful

Use this quick-start first:
docs/V0126_TESTER_QUICK_START.md

Use this copy-paste form when you are done:
docs/V0126_FEEDBACK_SUBMISSION_PACKET.md

Thank you. Plain honest reactions are more useful than polished feedback.
```
