# v0.128 Micro-Onboarding Spec

v0.128 adds a short contextual onboarding layer for the existing Salto review slice. It shows one compact prompt at a time and can be dismissed or privately skipped by automation.

## Sequence

1. Select Aster.
2. Move Aster to the quarry.
3. Capture and hold the quarry.
4. Send the Worker to mine or shrine posture.
5. Prepare for Ashen pressure.
6. Defeat the Ashen wave.
7. Restore the Lume link.
8. Review Results.

## Rules

- Show only one prompt at a time.
- Avoid prompt spam.
- Avoid developer jargon.
- Keep prompts compact enough to preserve battlefield readability.
- Keep private skip available for scripted validation without making routine Godot-editor work necessary.

## Constraints

This pass does not add broad progression, campaign systems, imported artwork, generated images, save changes, stable-ID changes, final engine selection, or full-port work.
