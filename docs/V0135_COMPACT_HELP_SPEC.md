# v0.135 Compact Help Spec

Status: `PASS_V0135_COMPACT_HELP` after the headed RTS ergonomics proof.

The optional help overlay is intentionally small and dismissible. It contains at most six lines:

- Select: left-click friendly.
- Box select: drag over squad.
- Move: right-click terrain.
- Attack: right-click Ashen.
- Zoom/pan: wheel plus WASD or arrows.
- Focus hero: Space.

F1 and the Help button open it. Escape dismisses it. The overlay is not required for routine play and does not replace visible objective guidance.
