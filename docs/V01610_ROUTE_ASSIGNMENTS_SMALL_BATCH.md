# v0.16.10 Route Assignments Small Batch

Date: 2026-05-22

Use these route labels for a 2-5 tester batch. Do not commit tester names, emails, or private contact details to the repo.

## Route A: Tutorial Only

- Start Tutorial / Proving Grounds.
- Play until completion, defeat, retry, or a clear blocker.
- Focus: onboarding clarity, basic command readability, Results flow.

## Route B: Tutorial Plus First Campaign

- Complete or exit Tutorial.
- Start the first campaign battle.
- Focus: whether Tutorial prepared the tester for the first real mission.

## Route C: Baseline Cautious

- Start a normal campaign route.
- Read objectives and warnings.
- Preserve units and retreat when badly hurt.
- Focus: normal first-session pacing and control clarity.

## Route D: Skirmish / Broken Ford

- Start a skirmish on Broken Ford if available from the current build.
- Focus: RTS controls without campaign progression pressure.

## Route E: Behaviour Mode Stress

- Select hero and units.
- Try Hold Ground, Guard Area, and Press Attack.
- Try attack hover, left-click attack, right-click movement, retreat, drag-select, minimap click, and `H`.
- Focus: whether control intent is readable and reliable.

## Assignment Suggestions

- 2 testers: Route A and Route E.
- 3 testers: Route A, Route B, Route E.
- 4 testers: Route A, Route B, Route C, Route E.
- 5 testers: Route A, Route B, Route C, Route D, Route E.
