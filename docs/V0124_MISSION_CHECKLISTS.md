# v0.12.4 Mission Checklists

Date: 2026-05-18

Use these checklists while playing. Short notes are better than perfect prose. If a question does not apply, write `N/A`.

For every mission, classify problems as one of:

- balance
- clarity
- controls
- UI density
- art debt
- actual bug

## 1. Tutorial / Proving Grounds

Checklist:

- Did I know what to do first?
- Did the tutorial objective and progress text help?
- Did selection/orders feel clear?
- Did movement feedback feel responsive?
- Did capture/resource/build/train steps make sense?
- Did combat feel readable?
- Did I understand what caused damage or losses?
- Did I know when to retreat or push?
- Did victory/exit feel fair?
- Did it clearly avoid campaign save/reward pressure?
