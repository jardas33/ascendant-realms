# v0.78 Creative Identity Lock Plan

Date: 2026-05-30

## Purpose

v0.78 is a docs-only strategic product-definition milestone. Its job is to make the long-term game identity clearer before the next runtime expansion begins.

The browser prototype remains valuable and stable. This pass does not change that prototype. It defines the intended destination: an original RTS/RPG world, a proposed public title direction, distinctive playable races, future hero architecture, visual governance, and a safe path from browser prototype to a later installable desktop game.

## Scope

Create a reviewable identity packet for Emmanuel covering:

- public title and brand options,
- Jardas, Salto, the Barrosan Marches, and Lume,
- the eight-race master roster draft,
- future Race + Class + Origin + Oath hero architecture,
- signature gameplay pillars,
- a five-act campaign outline,
- browser-to-desktop transition gates,
- visual direction and AI-art governance,
- display-name migration safety,
- original-IP separation,
- future implementation sequence,
- one concise Emmanuel review packet.

## Non-Goals

Do not implement gameplay. Do not alter runtime behavior. Do not rename runtime identifiers. Do not migrate saves. Do not generate, import, download, or wire art. Do not add assets. Do not start a desktop port, Electron wrapper, Tauri wrapper, engine switch, multiplayer code, PvP, co-op, new races, new maps, new units, new buildings, new classes, or broad UI changes.

## Internal Codename And Public Title

- Repository/internal codename remains `Ascendant Realms`.
- Leading public-title proposal for human review: `JARDAS: Oath of the Barrosan Marches`.
- `JARDAS` should be evaluated as the dominant logo word, with `Oath of the Barrosan Marches` as subtitle.
- No runtime rebrand is authorized in v0.78.

## Safety Rules

- Stable internal IDs remain unchanged.
- Save version remains unchanged.
- Current package/build/test flows remain intact.
- Package metadata may be updated only so the private playtest package can carry the v0.78 review docs.
- Legal and trademark review remain human responsibilities before public/commercial announcement.

## Required Review Outcome

v0.78 should end with Emmanuel able to answer:

- whether the title direction is correct,
- whether Jardas, Lume, Salto, and the Barrosan Marches have enough identity,
- which races feel memorable enough,
- which future race/class/visual slice should be prioritized,
- which runtime milestone should be allowed next,
- which decisions must remain deferred.

## Stop Rule

After v0.78 closeout, stop. Do not begin v0.79, runtime rebranding, art generation, desktop transition, or any proposed system implementation without a new explicit goal.
