# v0.124 HUD And Minimap Placeholder Spec

The player-facing HUD is compact and keeps the battlefield visible.

## HUD

- Resource row.
- Selected hero or entity card.
- Objective line.
- Command buttons: Move, Attack, Hold, Lume.
- Pause affordance.
- No developer rationale, test-step strip, or diagnostic jargon.

## Minimap

- Terrain-orientation panel.
- Friendly, hostile, objective, and Lume markers.
- Compact frame.
- No oversized debug card.

This is a placeholder HUD shell for review of density and posture, not final UI.
