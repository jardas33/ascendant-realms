# v0.124 Procedural Silhouette Spec

The player-facing slice replaces undifferentiated unit shapes with role-specific procedural silhouettes.

## Runtime Silhouettes

- Aster: capsule silhouette, larger selected scale, distinct hero color.
- Worker: compact box silhouette.
- Militia: six-sided cylinder silhouette.
- Ranger: narrow elongated box silhouette.
- Ashen enemy: tapered five-sided cylinder silhouette.
- Command Hall: base plus keep tower and banner silhouette.
- Barracks: paired training-wing silhouette.
- Mine/quarry, shrine, ruin, and Lume endpoint: procedural landmark silhouettes.

Selection discs and restrained health bars remain readable at RTS scale. No giant unit labels or final-art claims are introduced.
