# v0.100 Emmanuel Fast Review Guide

Target time: under 8 minutes.

## Launch

Use the latest private playtest package from `artifacts/playtest/` after package generation. Start it with:

```bash
node start-playtest-server.mjs
```

Open the local URL printed by the script.

## Fast Review

1. Open `Playtest Hub` from the Main Menu.
2. Confirm the top warning says private testing only and no rewards or campaign progress are saved.
3. Press `Run 8-Minute Visual Tour`.
4. Use `Open Step`, then return to the hub and continue through the tour.
5. Review Main Menu, Ascendant creation, Campaign map, Salto selected, selected hero, selected Worker, Lume first-link state, private-demo Results, Hero / Retinue, and return to hub.
6. Open `Campaign Shell -> Locked mission selected` and confirm the locked mission can be inspected but not launched.
7. Open `Lume -> Hidden`, `Auto`, and `Always` overlay samples if Lume readability still feels uncertain.
8. Open `Meta -> Ordinary Results` and `Meta -> Defeat Results`.
9. Press `Reset Private Preview`.
10. Return to Main Menu.

## Questions

- Is the hub easier than grinding to the same states manually?
- Is the no-save/no-reward warning clear enough?
- Are the groups named in a way that matches how you want to review the game?
- Does the 8-minute tour include the right screens?
- Which scenario is still missing or annoying to reach?
- Does the locked mission preview still feel safe?
- Are Lume Hidden / Auto / Always samples enough for quick visual review?
- Are ordinary and defeat Results fixtures useful without feeling fake?
