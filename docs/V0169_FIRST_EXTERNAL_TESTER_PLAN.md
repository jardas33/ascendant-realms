# v0.16.9 First External Tester Plan

Date: 2026-05-22

## Goal

Run a small 2-5 tester batch without leading questions and without adding private names to the repo. The aim is to find obvious control, onboarding, and readability blockers before broader testing.

## Batch Size

- Minimum: 2 testers
- Target: 3 testers
- Maximum: 5 testers

## Setup

- Use the clean private playtest package for the final v0.16.9 commit.
- Ask testers to use the included local server launcher, not `game/index.html`.
- Ask testers to record browser, OS, route, time played, and whether any issue repeated.
- Do not ask testers to validate a preferred answer.

## Routes

- Tutorial only
- Tutorial plus first campaign
- Baseline Cautious
- Skirmish / Broken Ford
- Behaviour mode stress

## What To Watch

- Do commands feel like they happen when intended?
- Is attack intent readable enough to use?
- Can testers retreat without feeling trapped by controls?
- Do Hold / Guard / Press make sense from observed behavior?
- Does Tutorial explain enough to start playing?
- Do Results screens help testers understand what happened?

## What Not To Judge Yet

- final art quality
- placeholder scale
- unfinished animation/VFX
- production balance
- worker construction
- future feature completeness
