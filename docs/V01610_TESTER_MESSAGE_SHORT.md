# v0.16.10 Tester Message Short

Date: 2026-05-22

Hi. This is a private release-candidate prototype of Ascendant Realms, a small browser RTS/RPG.

Please use the included launcher or local server helper. Do not open `game/index.html` directly.

Please play only your assigned route. Report what confused you, what felt responsive, what felt unresponsive, and where you got stuck. Do not focus on final art quality unless the visuals block understanding.

Please send back:

- assigned route
- browser and OS
- time played
- missions or modes played
- biggest confusion
- first control problem noticed
- whether attack, retreat, and selection felt clear
- whether Tutorial or Results explained what happened
- any repeated bug, with steps if possible
- screenshots or video only if useful

This is not a public build. Please do not repost the package or screenshots.
