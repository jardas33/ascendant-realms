# v0.83 Private Playtest Launch Notes

## How To Launch The Packaged Build

From the package folder:

```bash
node start-playtest-server.mjs
```

Then open the printed local URL, usually `http://127.0.0.1:4174/`.

## Aether Well Lume Demo

1. Start or continue a campaign.
2. Open Campaign Map.
3. Use `Launch Aether Well Lume Demo` in the private playtest tools row.
4. Capture linked sites on Broken Ford to test Linked Ward:
   - West Stone Cut + Ford Toll, or
   - Ford Toll + North Aether Spring.

## Isolation

This path is a demo shortcut only. Rewards, campaign progress, hero XP, Retinue, reputation, relic rewards, and optional-objective credit are disabled for the run.

## What To Judge

- Campaign map readability.
- Whether Aether Well Lume can be reached quickly from a fresh campaign.
- Whether the HUD and Results clearly explain the no-save/private-demo status.
- Whether Linked Ward is readable once the linked sites are captured.

## What Not To Judge

- Final art quality.
- Public campaign unlock pacing.
- New maps, factions, races, units, buildings, hero bindings, Worker bindings, Jardas bindings, desktop packaging, multiplayer, PvP, or co-op.
