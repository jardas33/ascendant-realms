# v0.165 Experimental Artifact Hygiene Inventory

Status: `DRY_RUN_ONLY`

## Tool

The inventory is generated by:

- `scripts/auditSaltoExperimentalArtifacts.mjs`
- `GODOT_AUDIT_SALTO_EXPERIMENTAL_ARTIFACTS_WINDOWS.bat`
- `npm run godot:audit:salto-experimental-artifacts`

The tool defaults to dry-run and writes:

- `artifacts/desktop-spikes/godot-salto/v0165/artifact-hygiene/salto-experimental-artifact-inventory.json`
- `artifacts/desktop-spikes/godot-salto/v0165/artifact-hygiene/salto-experimental-artifact-inventory.md`

## Classification

The audit classifies files into:

1. required tracked source/tooling/docs
2. required tracked diagnostic fallbacks
3. selected ignored local art sources
4. selected ignored local derivatives currently used by opt-in launchers
5. required local metadata
6. latest human-review captures
7. superseded local derivatives
8. superseded benchmark captures
9. stale temporary files
10. duplicate generated reports
11. disposable cache/build residue
12. unknown files requiring human review

## v0.165 Cleanup Action

No broad deletion is performed. The tool reports safe-to-delete future-approved candidates, archive candidates, and manual-review candidates. Unknown files fail closed.
