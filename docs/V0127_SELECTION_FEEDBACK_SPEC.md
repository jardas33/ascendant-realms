# v0.127 Selection Feedback Spec

v0.127 improves selection and order readability in the Godot player-facing Salto slice without changing gameplay rules.

## Feedback Contract

- Hover feedback uses a restrained focus ring for the captured subject.
- Click select shows role-specific selected markers for Aster and Worker.
- Box select shows squad markers without raw fixture labels.
- Enemy target, move-order, and attack-order markers are distinct procedural primitives.
- Health bars remain restrained and appear around selected or combat-relevant units.
- Damage flash and death fade are placeholder-only readability cues.

## Boundary

No debug labels, giant labels, imported assets, generated images, final UI claim, save write, stable-ID change, browser runtime change, final engine decision, or full-port work is authorized by this pass.
