## Recommendation

Begin with procedural shell hardening. A terrain material will not solve weak roads, river crossings, bridge silhouettes, site marker hierarchy, or block-like structure presentation by itself. Once procedural readability is stronger, run one bounded private terrain-material comparator before considering any player-facing material integration.
