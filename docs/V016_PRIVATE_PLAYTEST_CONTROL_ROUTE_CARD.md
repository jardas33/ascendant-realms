# v0.16 Private Playtest Control Route Card

Date: 2026-05-19
Audience: Emmanuel or a private tester with current package access

## Route

Use **Baseline Cautious** unless Emmanuel chooses a different route for comparison. The route should stay focused on controls, not balance tuning.

## First 10 Minutes

1. Open the private package through the included local server launcher.
2. Start Tutorial / Proving Grounds or First Claim.
3. Select the hero.
4. Confirm `Guard Area` appears.
5. Switch `Hold Ground`.
6. Hover and left-click a visible enemy.
7. Right-click away from combat.
8. Switch `Press Attack`.
9. Drag-select across the HUD and minimap.
10. Press `H` and confirm the hero panel refreshes.

## Deeper Route

1. Baseline Cautious route.
2. One skirmish/control check.
3. One tutorial win/loss check.
4. Behaviour modes with the hero and at least one group.
5. Minimap/HUD stress after using behaviour controls.
6. Attack intent stress: hover enemy, move away, hover again, left-click.
7. Melee contact stress: hero or unit beside a hostile target without nudging.
8. Retreat stress: repeated move-away commands while enemies remain nearby.

## What To Report

- Snap-back loop after retreat.
- Idle contact melee.
- Attack hover or left-click mismatch.
- Mode copy unclear or not updating.
- Stale HUD after minimap/HUD interactions.
- Tutorial victory/defeat Results routing problem.

## What Not To Tune From One Run

- `Press Attack` being too aggressive.
- `Hold Ground` feeling boring.
- One odd movement path that does not repeat.
- Balance opinions about unit strength or economy.
