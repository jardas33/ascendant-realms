## Candidate Reading

T1, `v0144-silhouette-t1-balanced-barrosan-readability`, is the best overall convergence candidate. It should be judged for combined readability, grounded Barrosan materials, Worker support tools, and fast role separation.

T2, `v0144-silhouette-t2-builder-support-clarity`, is the Worker-role clarity candidate. It should be judged for whether Worker reads as Builder / repair support / site support rather than miner-only.

T3, `v0144-silhouette-t3-commander-champion-restraint`, is the Aster Commander / Champion restraint candidate. It should be judged for hero presence without weapon-dominant identity. The visible long blade is a review concern, not an approval.

## Review Questions

- Which T board best converges toward a reference-only Aster / Worker silhouette-scale lock?
- Which board makes Aster readable as Commander / Champion without making a weapon central?
- Which board makes Worker read as versatile Builder, repairer, and site-support unit rather than miner-only?
- Which gameplay-distance inset best proves role separation on wet-granite / worked-earth terrain?
- Which board leaves the most practical room for future selection rings and markers?
- Does any board lock unsupported race, class, final costume, final weapon, final face, final faction insignia, or final production detail?
- Does any board resemble a protected game, famous character, generic mobile-game style, or protected worker/hero archetype too closely?
