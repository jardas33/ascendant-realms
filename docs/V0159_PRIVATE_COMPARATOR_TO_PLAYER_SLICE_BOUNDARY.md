# V0159 Private Comparator To Player-Slice Boundary

Status: boundary statement for v0.159. This checkpoint prepares the crossing plan but does not cross it.

## Boundary Interpretation

Private comparator success means the hybrid-art path is technically promising. It does not mean the art is production-approved, player-facing-approved, or ready for broad runtime integration.

v0.158 proved a selected five-slot private comparator posture under stress. v0.159 records that evidence and prepares a future one-slot opt-in Worker contract. It does not move any selected candidate into the normal Salto player slice.

## Allowed In v0.159

- Create readiness documentation.
- Create the first-slot scorecard.
- Create the future v0.160 Worker opt-in integration contract.
- Create risk, rollback, boundary, review, and implementation docs.
- Create a future v0.160 prompt for human review.
- Add documentation validation proving the packet exists and the default launchers remain unmodified.

## Forbidden In v0.159

- Generate AI images.
- Add runtime-art slots.
- Add opt-in launchers.
- Import selected private comparator art into the normal Godot player slice.
- Import any selected private comparator art into the browser runtime.
- Modify default Salto launchers.
- Mutate production manifests, package manifests, or art-slot registries.
- Change saves, stable IDs, gameplay, controls, AI, objectives, maps, balance, or browser behavior.
- Start v0.160.

## Future Crossing Rule

Only a future explicit v0.160 bounded goal may cross this boundary, and only for `worker_billboard_static_v0147` through a Godot-only opt-in launcher with fail-closed procedural fallback.
