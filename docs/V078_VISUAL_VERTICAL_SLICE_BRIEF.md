# v0.78 Visual Vertical Slice Brief

Date: 2026-05-30

## Status

Future brief only. Do not generate, import, download, or wire assets in v0.78.

## First Polished Visual Slice

The first human-reviewed visual slice should include:

- Salto village environment,
- one Barrosan hero,
- one Barrosan Worker,
- two Barrosan military units,
- one Command Hall,
- one Barracks,
- one mine,
- one shrine,
- one Ashen enemy unit,
- one coherent HUD frame,
- one battlefield terrain style frame,
- one representative combat moment,
- one representative Lume-site moment.

## Concept-First Assets

Start as static concept/style-frame work:

- Salto mood frame,
- Barrosan hero silhouette sheet,
- Barrosan Worker/basic/ranged troop silhouettes,
- Barrosan Command Hall and Barracks architecture sheet,
- mine and shrine landmark sheets,
- Ashen enemy silhouette sheet,
- HUD frame direction,
- battlefield material sheet.

## Later Runtime Assets

These later require sprites, models, animation, VFX, UI integration, or engine-specific rebuilding:

- walk/attack/work animations,
- building construction states,
- damage states,
- Lume-site activation effects,
- projectile/impact effects,
- command icons,
- terrain tiles/materials,
- audio and ambience.

## AI-Assisted Ideation Candidates

AI-assisted ideation may be useful for:

- mood thumbnails,
- material studies,
- non-final silhouette exploration,
- architecture variation,
- color/lighting studies.

These outputs must remain reference-only until human review approves source, license, originality, consistency, and production suitability.

## Human Review Required

Human review is mandatory before runtime use for:

- final character/unit designs,
- faction architecture,
- icons,
- UI frames,
- animation keyframes,
- VFX language,
- audio style,
- any asset that might resemble protected expression.

## Consistency Enforcement

- one approved style-frame set,
- race-specific anatomy sheets,
- architecture sheets,
- scale chart,
- color/material guide,
- visual manifest metadata,
- screenshot QA targets,
- package validator protection against uncontrolled assets.

## Not Approved In v0.78

No art generation, no asset import, no runtime replacement, no engine switch, no desktop port, no UI redesign, and no visual overhaul.
