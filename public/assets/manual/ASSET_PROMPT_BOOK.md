# Ascendant Realms Manual Asset Prompt Book

This file is generated by `npm run assets:prompts`.

## How To Use

1. This prompt book is for manual image generation only.
2. Do not paste API keys anywhere. No API image generation is required.
3. Copy one MAIN PROMPT and one NEGATIVE PROMPT into ChatGPT image generation.
4. Generate one asset at a time and reject results that contain text, watermarks, copied symbols, or unclear silhouettes.
5. Download the image, rename it exactly as instructed, and place it in the listed folder.
6. After adding images, run npm run assets:manifest and npm run assets:validate.
